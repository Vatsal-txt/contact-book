# 📇 Contact Book Project

## 📌 Overview
This project is a simple Contact Book application implemented in two parts:

- 💻 C++ Console Application
- 🌐 Web-based Interface (HTML, CSS, JavaScript)

It allows users to manage contacts by adding, searching, deleting, undoing deletions, and exporting data.

---

## 🚀 Features

### 🔹 C++ Application
- Add new contacts
- Display all contacts
- Search contact by name
- Delete contact
- Undo last delete (Stack implementation)
- Export contacts to JSON file
